# Assets Folder

This folder holds all visual assets for the Bolo Boys website.

## To add assets:
Upload the following from Google Drive and drop them here:

- **Logo** — `logo.png` or `logo.svg` (SVG preferred for crisp scaling on all screens)
- **Band photo (hero)** — `hero.jpg` — wide landscape shot, used as the top banner background
- **Band photo (bio)** — `bio.jpg` — optional, used alongside the bio section

Once files are here, let Claude know and it will wire them into the site.
